    Seccion de apuntes de la materia

    Introduccion a la programacion

    Para empezar en el mundo de la programacion debes saber como se constituye un algoritmo, es una secuencia que tiene un inicio y un fin y realiza un determinado número de pasos para cumplir su objetivo, es necesario practicar la logica ya que esto te facilitara a que puedas resolver los problemas de manera mas rapida asi que empecemos con lo basico

    Tipos de variables

    Las variables constan de dos tipos las primitivas y las no primitivas,entre las variables primitivas tenemos a los numeros, caracteres y los booleanos, entre las no primitivas tenemos la variable cadena.

    Variables numericas

    existem varios tipos de variables numericas dependiendo a el tipo de numeros y el tamaño que estos representan entre las mas conocidos tenemos a Integer y Double; Integer en este tipo de variable se le puede asignar valores enteros y su syntaxis de asignacion es int, Double es el tipo de variable en el que se le asigna valores quebrados su sintaxis de asignacion es double

    int numero = 10; asi es como se le asigna un valor con enteros

    double quebrado = 5.2; asi es como se asigna un valor con enteros

    Cabe aclarar que al momento de crear variables es la siguiente TipoDeVariable Nombre = Valor; es importante poner el ";" al terminar cada linea siempre y cuando no se trate de que haya "}" delante.

    Variables de tipo caracter

    Este tipo de variables almacenan un caracter sin importar el tipo de caracter que sea , este tipo de variable es Character y su sintaxis de asignacion es char para la asignacion el valor que le queremos dar debe estar entre comillas simples '' .

    char letra = 'a';

    Variable booleana

    Esta variable solo se le puede asignar dos valores true o false, esto nos sirve para identificar el estado de alguna variable su sintaxis de asignacion es boolean.

    boolean alerta= true;

    Variable cadena

    Este tipo de variable es considerado cadena por que se le puede asignar varios caracteres es decir es una cadena de chars su sintaxis de asignacion es String y para utilizarlo el valor debe estar entre comillas dobles "".

    String palabra= "infopedia";

    Metodos

    Para empezar aprenderemos como se hace un metodo, este consta de cuatro partes estado del metodo, tipo de variable que devuelve, nombre del metodo y parametro que necesita

    Estado del metodo: private (significa que esto no sera visible para el usuario), public (significa que esto sera visible para todos).

    Tipo de variable que devuelve: el metodo que realices puede devolver cualquier cosa que se lo pidas pueden ser int, char, boolean , String ,etc; pero tambien es posible hacer un metodo que no devuelva nada para eso utilizamos la palabra void.

    Nombre del metodo: podemos asignarle cualquier nombre al metodo que realices pero lo mas recomendable es asignarle como nombre la operacion que realiza por ejemplo si tu metodo suma entonces podrias ponerle como nombre Sumador

    Parametros: los parametros van dentro de parentesis y separados por comas de uno uno en uno, en esta parte asignamos variables que necesitemos para realizar el metodo.

    Al finalizar un metodo se ponen llaves{ }y todo el contenido de lo que hace el metodo va dentro de las llaves.

    public int Sumar(int n1,int n2){ aqui va el algoritmo}

    Imprimir consola

    Para impromir pantalla solo debemos poner dentro del metodo que realices System.out.println() entre los parencesis puedes asignar lo que gustes imprimir, si quieres imprimir texto debes ponerlo entre comillas dobles como si se tratara de un String o tambien podrias imprimir variables para ello simplemente pones el nombre de la variable que quieras imprimir , tambien es posible intercalar entre texto y variables para ello debemos usar la concatenacion que consiste poner el signo + entre una variable y un texto.

    System.out.println("texto"+variable+","+variable);

    Condicional

    Las condicionales nos sirven para restringir el metodo para que realicen una tarea u otra, para ello utilizamos if() dentro de los parentesis va la condicion que se debe cumplir para entrar a esa parte del metodo que debe estar entre {}, si no cumple esa condicion entonces ignora esa parte del metodo pero if() tiene un complemento que es el else esto nos sirve para decir si no cumple el if() entonces quiero que haga algoque esta dentro de las {} posteriores al else

    int x=2

    if(x<3){< /p>

    x=x+2;

    } else{

    x=0;

    }

    Iteradores

    Los iteradores son ciclos que realizan una tarea repetitiva hasta que el ciclo acabe mencionaremos dos el for y el while

    For()

    for requiere 3 parametros que son un contador, una condicion y de que manera se incrementara la variable los tres parametros deben estar separados por punto y coma ";" fuelgo todo lo que pongas dentro de las llaves se repetira hasta que la condicion se cumpla

    int numero=0;

    for(int i=0; i<10;i++){< /p>

    numero = numero +2;

    } (al poner el ++ quiere decir que a ese nuero se le aumentara 1 y si ponemos -- signigica que disminuira en 1)

    while()

    Este iterador solo require un parametro que es un condicional, todo el metodo dentro las llaves posteriores al while se repetira hasta que la condicion se cumpla.

    int numero=0;

    while(numero=10){

    numero++;

    }

    Vectores

    Los vectores son un conjunto determinado de variables del mismo tipo que tiene un numero finito es como un casillero donde cada espacio puede ser ocupado por una variable y solo se puede poner un numero determinado de variables para asignar vectores se pone:

    int []numeros= new int[10];

    Se asigna el tipo de variables que queremos almacenar seguino de corchetes esto hace referencia a que sera un vector seguido pones el nombre que quieres que lleve el vector, el valor que se le asigna es la palabra new que significa nuevo seguido del tipo de variables a almacenar y entre corchetes la cantidad de variables que se podra guardar, al momento de crearse el vector en el caso de enteros las variables adquiriran el valor de 0 lo que quiere decir que nunca estaran vacios,para cambiar el valor de alguna de las variables almacenadas se hace lo siguiente:

    numeros[5]= 4;

    Esto quiere decir que la variable almacenada en el espacio 5 tendra el valor de 4, cabe aclarar que las variables se cuentan desde 0, por ello el espacio maximo que tiene el vector del ejemplo anterios es el espacio 9

    Tambien es posible saber el tamaño de un vector para eso utilizaremos el nombre del vector seguido de ".length":

    numeros[].length;

    ArrayList()

    Los ArrayList al igual que los vectores nos sirven para almacenar variables pero la diferencia es que no debemos asignar un tamaño al ArryList esto es bastante util al momento de almacenar un numero indeterminado de variables para crear un ArrayList se hace de la siguiente manera:

    ArrayList<Integer>numeros = new ArrayList();

    En este caso tambien se utiliza la palabra new para crear el ArrayList ponemos paretesis sin nada por que no es necesario especificar su tamaño u otra cosa.Al momento de crear un ArrayList este se crea vacio y te preguntaras como añadimos elementos a esta ista pues de la siguiente manera:

    numeros.add(4)

    Utilizamos el nombre de la lista seguido de ".add" esto significa que añadiremos un nuevo elemento y entre parentesis va el elemento que queremos añadir dependiendo que tipo de elemento queramos darle se le tiene que pasar de distintas maneras es decir si es un numero se le hace sin ningyuna distincion si es un char se pone en '' y si es un String en "" pero en si se puede hacer listas de todo incluso de objetos pero eso lo veremos mas adelante. Si queremos saber el valor de algun elemento basta con llamar a su posicion esto se hace de la siguiente manera:

    numeros.get(0);

    Esto te devolvera el valor que hay almacenado en esa posicion de la lista, para recorrer la lista puedes utilizar elementos enseñados anteriormente como el for y el while pero eso es cosa de tu ingenio, tambien es posible eliminar un elelmento de la lista, basta con saber la posicion que ocupa y se hace de la siguiente manera:

    numeros.remove(5)

    Utilizamos el nombre de la lista seguido de ".remove()" entre parentesis va el indice o posicion de lo que queremos quitar de la lista. como ya lo viste antes utilizamos el "." y ponemos algo que queremos realizar pero ¿para que sirve el punto?. Pues este punto nos sirve para acceder a los metodos que tienen algunas variables u objetos.

    Objetos

    ¿Que es un objeto? practicamente puede ser cualquier cosa, pero se define como la representacion abstracta de algo. Para crear un obtejo solo debemos pensar que queremos crear, que uso le daremos y que atributos tendra, los atributos son variables que definen el objeto puede tener las que gustes pero debes considerar si los atributos que asignes sirven para algo en tu programa

    Empecemos por definir un objeto llamado casa que tendra como atributos: precio, numero de habitaciones y numero de pisos:

    private int Precio;

    private int Nhabitaciones;

    private int Npisos;

    Usamos la palavra private, esto quiere decir que nuestros atributos seran de tipo privado que solo el objeto tendra control sobre ellos,seguidamente ponemos el tipo de variables que queremos asignar y luego el nombre que recibe ese atriuto, luego hacemos el constructor lo que hace este constructor es asignarle los valores a un objeto.

    public Casa(int precio, int habitaciones,int pisos){

    Precio=precio;

    Nhabitaciones=habitaciones;

    Npisos=pisos;

    Utilizamos el estado publico para poder crear un objeto, seguido pones el nombre del objeto que estas creando y entre parentesis se pone los parametros quese necesita para la creacion de ese objeto y de esta manera se crea un objeto, al momento de crear un objeto para cambiar el valor de algun atributo o mostrar algun atributo se utilizan los metodos set y get.

    Metodo set

    Este metodo lo que hace es cambiar el valor de algun atributo por otro para ello creamos asi el metodo:

    public void setPrecio(int precionuevo){

    Precio=precionuevo;

    }

    De esta manera se cambia un atributo, debes tener cuidado que al momento de cambiar debes darle un stributo del mismo tipo de variable.

    Metodo get

    El metodo get sirve para mostrar el valor de un aatributo para ello no necesitamos mandarle ningun parametro y se hace de la siguiente manera:

    public int getPrecio(){

    return Precio;

    }

    ponemos el estado publico para que pueda acceder cualquier persona y el tipo de variable que es el atributo seguidp de la palabra get junto con el nombre de la variable y poner parentesisi vacios ya que no requiere ningun parametro ,mostramos el valor con return seguido del nombre del atriburo.Asi de simple es crear un objeto espero les haya gustado esta pequeña introduccion a la programacion y que les haya sido util :3
